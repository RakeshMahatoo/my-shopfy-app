# 🛍️ Shopify Admin App

A Shopify embedded admin app built with Remix, React, and Shopify GraphQL API.

## 🚀 Features

- View all products with images, price, status, inventory
- Create new products via form
- Real-time stats — Total, Active, Inventory count
- Secure GraphQL mutations with variables
- Built with Shopify App Bridge

## 🛠️ Tech Stack

- **Remix** — Full stack React framework
- **Shopify App Bridge** — Embedded app authentication
- **Shopify Admin GraphQL API** — Product management
- **React** — UI components
- **Shopify CLI** — App development

## ⚙️ Setup

1. Clone the repo
```bash
   git clone https://github.com/YOUR_USERNAME/shopify-remix-app.git
   cd shopify-remix-app
```

2. Install dependencies
```bash
   npm install
```

3. Connect to your Shopify store
```bash
   shopify app dev
```

## 📡 API Used

| API | Purpose |
|-----|---------|
| `products` query | Fetch all products |
| `productCreate` mutation | Create new product |
